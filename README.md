##  EasyMarket 本地部署

> 1.  将nideShop.mysql导入您本地的数据库中
> 2. 更改数据库的配置文件目录（src/common/config/database.js）
> 3. 将[EasyMarketApp](https://github.com/Peroluo/easyMarketApp)  打包文件放到 （www/web/easyMarket目录下）
> 4. 安装依赖 cnpm install
> 5. 启动项目 npm start
> 6. 浏览器访问 http://127.0.01.1:8888/web/easyMarket
